	************************************
*****  	Description de l'application:   	*****
	************************************

ProxiBanqueSI est une application permettant de gérer les comptes et les clients d'un ensemble d'agences bancaires de ProxiBanque.


	************************************
*****  		Fonctionnalités:		*****
	************************************

L'application permet de :
- ajouter, modifier, lire les informations et supprimer des clients
- gérer l'effectif de clients de chaque conseiller
- réaliser des virements de comptes à comptes (intra- et inter- clients)
- auditer l'agence pour vérifier l'état des comptes des clients
	
	************************************
*****  		Livrables:			*****
